# Brandisby

An e-commerce platform that allows creators to launch their own branded stores.

## Structure
- **Homepage**: `index.html`
- **Fleur De Vie Store**: `/FleurDeVie`
- **Serac Store**: `/Serac`
- **Global Styles**: `/styles`
- **Scripts**: `/scripts`
- **Product Data**: `/data`

## Deployment
- Hosted on Vercel
- Version control with GitHub
